Extração de Dados Lattes

O CNPq disponibiliza às instituições a possibilidade de extração de dados da Plataforma Lattes, tanto da base de Currículos Lattes como do Diretório dos Grupos de Pesquisa, por meio de uma ferramenta chamada Lattes Extrator. Para saber todos os procedimentos e ter acesso aos formulários para a instituição efetuar cadastro e solicitar acesso ao Lattes Extrator, o interessado deverá encaminhar pedido a atendimento@cnpq.br.

Requisitos:

  • 1) Ser Instituição de Ensino e Pesquisa.
  • 2) Possuir um IP dedicado.
  • 3) Desenvolver solução tecnológica para integração com o Webservice do Lattes Extrator.

Número Identificador - ID dos Currículos Lattes
